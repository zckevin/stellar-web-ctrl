# webtorrent-stellarplayer-plugin

Export control endpoint through web api.

## Web API

### /play_url

play video with HTTP url

method: POST

using HTTP form data, "application/x-www-form-urlencoded"

```JavaScript
{
  url: "http://..."
}
```

## Default binding port: 28080
